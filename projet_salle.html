Application de réservation de salles

DEV

Pour un projet fictif j'ai développé une application de réservation des salles en ligne pour une organisation.
Cette application embarque une base de données SQLite, et une gestion des authentifications. Elle est développée en Golang avec le framework Gin pour le back-end, en HTML, CSS et JavaScript pour le front-end, et déployée à l'aide d'un Dockerfile sur : https://app-reservation.onrender.com/
